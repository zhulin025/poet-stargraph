# 苏轼朋友圈 3D 知识图谱 - 产品需求文档

**版本**: v1.0  
**更新日期**: 2026-03-16  
**项目类型**: 3D 交互式知识图谱可视化  

---

## 📋 项目概述

### 项目定位
基于 3D 力导向图技术，可视化展示苏轼及其朋友圈的社交关系网络，通过交互式 3D 视觉效果呈现宋代文人的社交生态。

### 核心价值
- **文化传播**: 以现代化可视化方式呈现古典文学关系
- **学术研究**: 辅助宋词研究者理解文人社群结构
- **教育科普**: 降低古典文学理解门槛，增强趣味性

---

## 🛠️ 技术栈

### 核心渲染
| 技术 | 用途 | 选型理由 |
|------|------|----------|
| **Three.js** | WebGL 3D 渲染 | 成熟的 WebGL 封装，性能优秀 |
| **d3-force-3d** | 物理力学模拟 | 灵活的力导向算法，可定制性强 |
| **3d-force-graph** | 高层封装库 | 快速构建 3D 图谱，减少样板代码 |
| **Vue 3 / React** | UI 框架 | 组件化开发，状态管理便捷 |

### 数据处理
| 技术 | 用途 | 说明 |
|------|------|------|
| **NLP 关系抽取** | 宋词关系挖掘 | 识别赠诗、唱和、引用等关系 |
| **Python + spaCy/ jieba** | 文本预处理 | 分词、实体识别 |
| **Neo4j / JSON** | 图数据存储 | 灵活选择存储方案 |

```javascript
const haloGeometry = new THREE.SphereGeometry(nodeRadius * 1.8, 32, 32);
```

## 📊 功能需求

### 核心功能（P0）

| 功能 | 描述 | 验收标准 |
|------|------|----------|
| **3D 力导向布局** | 节点自动排布，避免重叠 | 初始加载后 3 秒内稳定 |
| **发光节点** | 所有节点带发光效果 | 暗色背景下清晰可见 |
| **光晕效果** | 节点外围半透明光晕 | 光晕不遮挡相邻节点 |
| **粒子连线** | 连线上有流动粒子 | 粒子流畅循环运动 |
| **自动旋转** | 场景缓慢自转 | 用户可手动暂停 |
| **搜索诗人** | 输入姓名定位节点 | 搜索后相机聚焦+高亮 |
| **朝代切换** | 筛选唐/宋诗人 | 切换后图谱平滑过渡 |
| **图例面板** | 显示颜色/关系说明 | 可折叠，不遮挡主视图 |
| **控制按钮** | 旋转/翻转/布局重置 | 按钮响应<100ms |

### 增强功能（P1）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **诗人详情卡片** | 点击节点显示生平/作品 | P1 |
| **关系筛选** | 按关系类型过滤连线 | P1 |
| **时间轴滑块** | 按年份动态展示关系演变 | P1 |
| **导出图片** | 导出当前视角高清截图 | P2 |
| **VR 模式** | 支持 VR 头显沉浸式浏览 | P3 |

```json
      "death": 1101,

      "works_count": 2700,
      "description": "字子瞻，号东坡居士..."
    }
  ],
  "links": [
    {
      "source": "su_shi",
      "target": "su_zhe",
      "type": "兄弟",
      "weight": 5,
      "evidence": ["《水调歌头·明月几时有》序"],
      "year": 1076
    }
  ]
}
```

### 影响力计算公式
```javascript
influence = (mentioned_count * 3) + (works_count / 10) + (relation_count * 2)

// 示例：苏轼
// mentioned_count = 200 (被提及次数)
// works_count = 2700 (作品数)
// relation_count = 50 (关系数)
// influence = 600 + 270 + 100 = 970
```

## 🎬 交互设计

### 用户操作流程

```
1. 打开页面 → 自动加载 3D 图谱（苏轼居中）
2. 浏览探索 → 拖拽旋转、缩放查看
3. 搜索诗人 → 输入框输入姓名 → 相机聚焦
4. 查看详情 → 点击节点 → 弹出详情卡片
5. 筛选过滤 → 选择朝代/群体 → 图谱动态更新
6. 导出分享 → 点击导出 → 下载高清图片
```

### 交互反馈

| 操作 | 视觉反馈 | 时长 |
|------|----------|------|
| **Hover 节点** | 节点放大 1.2x + 显示姓名 | 即时 |
| **Click 节点** | 相机平滑移动 + 弹出卡片 | 500ms |
| **搜索定位** | 路径高亮 + 节点脉冲 | 800ms |
| **朝代切换** | 节点淡入淡出 | 300ms |
