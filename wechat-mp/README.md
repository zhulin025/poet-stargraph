# 诗外星辰 - 微信小程序壳工程

这个目录是一个独立的小程序项目，通过 `web-view` 嵌入了诗人星图的 H5 版本。

## 快速开始

1. **下载微信开发者工具**：
   安装并运行 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。

2. **导入项目**：
   在开发者工具中选择“导入项目”，目录选择当前这个 `wechat-mp` 文件夹。

3. **配置 AppID**：
   在该目录的 `project.config.json` 中，或者开发者工具的“详情”面板里，将 `appid` 修改为您在 [微信公众平台](https://mp.weixin.qq.com/) 申请的真实 AppID。

4. **更新 H5 链接**：
   打开 `pages/index/index.js`，将 `url` 变量修改为您正式部署后的 H5 网址。

## 重要：域名校验

为了让小程序能正常访问您的 H5 页面，您必须：
1. 在微信小程序后台“开发设置 -> 业务域名”中添加您的 H5 域名。
2. 下载微信提供的校验文件（通常是一个 `.txt` 文件）。
3. **将该校验文件放在主项目的 `public/` 目录下**。
4. 重新部署主项目，确保通过 `https://您的域名/校验文件名.txt` 能正常访问。

## 注意事项
- 由于使用了 `web-view`，请确保您的 H5 项目已适配移动端触摸。
- 只有已备案的 HTTPS 域名才能被添加为业务域名。
