# 诗外星辰 (Verses & Voids) V4.0 旗舰美学版

> **“以星辰之势，重绘千年文人星图。”**

`诗外星辰` 是一个基于 **多巴胺波普 (Dopamine Pop)** 美学设计的 3D 交互式文人关系时空观测站。本项目利用 3D 引力图可视化技术，将散落在浩瀚史料中的诗人关系（从盛唐到大清）重构为动态的社交星云，旨在为古典文学的研究与欣赏提供一个充满活力、极具现代感的视角。

## 💎 项目定位与价值

### 1. 核心定位

不同于传统的静态图谱，`诗外星辰` 致力于构建一个 **“活着的文学宇宙”**。它不仅是数据的可视化，更是一个可以交互、可以感知的时空剧场。

### 2. 核心价值

- **文化传播**：通过潮流的 3D 视觉语言，打破古典文学的沉闷感，吸引年轻群体探索传统文化。
- **关联研究**：辅助研究者直观观察不同朝代、不同流派（如苏门四学士、蜀党、变法派）的社群拓扑结构。
- **沉浸教育**：通过极简的操作与直观的视觉反馈，显著降低了理解古代复杂人际网的门槛。

## 📊 数据逻辑与采集处理

### 1. 数据来源

项目数据依托于以下公开资源进行深度挖掘：

- **古诗文网**：采集诗人基础生平、传世作品数及名篇内容。
- **维基文库/全宋词/全唐诗**：提取诗人间的赠答、唱和、引用关系。
- **中国哲学书电子化计划**：校验人物生卒年及地理迁徙轨迹（Spatiotemporal data）。

### 2. 智能化处理流程

1. **实体识别 (NER)**：从诗歌题目（如《赠...》、《酬...》）和正文中识别特定人名。
2. **关系分类**：利用 NLP 技术将互动归类为：好友、师徒、政敌、同门、时空交集等。
3. **影响力建模**：我们设计了一套科学的影响力计算公式：  
   `Influence = (被提及次数 * 3) + (作品总数 / 10) + (社交关系数 * 2)`

## 🎨 视觉架构设计

### 1. 节点设计 (Nodes)

- **层级视觉**：节点半径与影响力（val）呈对数增长关系，顶级诗人（如苏轼、李白）拥有显著的内核。
- **光晕系统**：节点外围包裹着一层根据朝代颜色渲染的半透明光晕，模拟恒星的能量场。
- **流派配色**：
  - **金色群落**：如苏门四学士等核心名士圈。
  - **红/蓝对比**：代表不同的政见派系（如变法派与保守派）。

### 2. 连线设计 (Links)

- **社交能量场**：连线不仅仅激是线条，我们利用 **粒子流动 (Directional Particles)** 表现社交互动的强度与方向。
- **同时代感知**：对于生卒年有交集但缺乏明确史料记载互动的诗人，引入了“同时代”细连线，还原真实的历史背景。

## 🚀 性能优化与技术方案

为了在 Web 端（尤其是移动端）实现极致顺滑的 3D 体验，我们实施了多项优化：

- **纹理智能缓存 (Texture Caching)**：针对高密度的 3D 文本标签（KaiTi 楷体渲染），通过 Canvas 纹理缓存极大地降低了 GPU 负担。
- **自适应渲染机制**：移动端自动降低粒子密度并启用简化的节点光晕效果，确保 30fps+ 的流畅操作。
- **数据集延迟加载**：采用分朝代数据的动态 import 机制，首屏加载包体减小了约 700KB。
- **Canvas 持久化技术**：启用 `preserveDrawingBuffer`，确保在任何复杂视角下都能导出高清带水印的快照。
