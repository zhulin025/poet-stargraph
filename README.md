# 诗外星辰 (Verses & Voids) V2.0 简体版

**探索诗词背后的社交宇宙。**

一个基于多巴胺波普风格 (Dopamine Pop Style) 的 3D 文人关系时空观测站。利用 3D 引力图可视化从盛唐到大清历代诗人的社交网络、影响力矩阵及传世名篇。

![项目截图](public/screenshot.png)
![功能展示](public/screenshot2.png)

## ✨ 核心特性

- **多巴胺视觉系统**：采用高饱和度色彩空间、2px 精品化粘土投影与现代波普风 UI 界面。
- **3D 关系星系**：基于 `react-force-graph-3d` 实现的星系引力场，动态计算“同时代”潜在关联。
- **古典作品全读**：支持点击诗人卡片查阅代表作，内置**古典竖排排印**全屏弹窗。
- **全朝代覆盖**：深度整合唐、宋、元、明、清五个核心朝代的诗人生平与作品数据。
- **沉浸式交互**：支持节点点击对焦、镜头自动旋转、hover 自动暂停等丝滑交互。
- **移动端特化**：深度优化小屏交互与视野占比，支持全屏控制台遮罩。

## 🛠 技术栈

- **框架**: Next.js 14 (App Router)
- **渲染**: Three.js / React Force Graph 3D
- **样式**: Tailwind CSS (Dopamine Theme)
- **排印**: Noto Serif SC (古典衬线体)
- **动画**: Framer Motion
- **图标**: Lucide React

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 手机端预览 (局域网)
# 假设本机 IP 为 192.168.31.119
npm run dev -- -H 0.0.0.0
```

## 🌐 部署

项目支持 GitHub Pages 自动部署。构建结果将输出为静态 HTML 站点。
