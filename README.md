# 诗外星辰 (Verses & Voids) V4.0 旗舰美学版

**探索诗词背后的社交宇宙。**

一个基于多巴胺波普风格 (Dopamine Pop Style) 的 3D 文人关系时空观测站。利用 3D 引力图可视化从盛唐到大清历代诗人的社交网络、影响力矩阵及传世名篇。

## ✨ V4.0 新特性 (Release 2026-03-18)

- **全场景主题适配**：手机端控制台现在完美适配白昼/黑夜模式，确保全天候的操作一致性。
- **旗舰级导出系统**：
  - **动态排印**：诗歌导出功能现支持根据文本长度自动调整字号，配合全新设计的古典金边框，美学效果大幅提升。
  - **3D 连线增强**：大幅增强了导出图片中“同时代”连线的清晰度，完整记录时空交织。
- **古典沉浸架构**：
  - **智能视口**：移动端长诗词自动滚动至起始位置，诗人卡片关闭按钮固定，操作更显丝滑。
  - **纯净视野**：桌面端优化了顶部交互区域，隐藏冗余控制，让感知完全聚焦于千年的名士星云。
- **楷体渲染与 4x 高清**：重制 3D 姓名标签采样逻辑，支持 **512x128** 超高清分辨率，并引入经典的 **“楷体 (KaiTi)”** 字体序列，呈现出兼具锐利度与书法底蕴的 3D 视觉。
- **纹理智能缓存 (Texture Caching)**：针对大宋等极密集节点场景，实施了底层显存优化，通过缓存 Sprite 材质极大提升了 3D 拖拽与缩放的顺滑感。

## 🛠 核心技术栈

- **框架**: Next.js 14 (App Router)
- **渲染**: Three.js / React Force Graph 3D (启用 `Texture Cache` & `preserveDrawingBuffer`)
- **样式**: Vanilla CSS (Dopamine Clay Style)
- **排印**: Noto Serif SC & KaiTi (古典竖排排印系统)
- **逻辑**: 数据驱动挖掘 & 动态关联索引
- **动画**: Framer Motion & CSS keyframes

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
```
